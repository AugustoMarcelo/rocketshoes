## ðŸ¤” How to run on my machine?

```bash
  # download the project
  git clone https://github.com/AugustoMarcelo/rocketshoes.git

  # accessing the project folder
  cd rocketshoes

  # download the project dependencies
  yarn

  # running fake server
  yarn api:server

  # start application
  yarn start
```
